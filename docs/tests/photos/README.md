# Test Photos

Place coffee cup photos here for E2E testing.

## Required Photos

| File | Purpose |
|------|---------|
| coffee-cup-1.jpg | Standard coffee grounds photo |
| coffee-cup-2.jpg | Second coffee pattern |
| coffee-cup-dark.jpg | Dark/underexposed photo |
| coffee-cup-blurry.jpg | Slightly blurry photo |
| not-coffee.jpg | Non-coffee image (cat, food) |
| large-photo.jpg | >10MB file (for size limit test) |

## Photo Requirements

- JPEG or PNG format
- Coffee cup with visible grounds pattern
- Various lighting conditions
- At least 800x800 pixels recommended
