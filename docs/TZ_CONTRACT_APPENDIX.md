## 1. ОПИСАНИЕ ПРОЕКТА

Разработка AI-платформы для психологического анализа кофейной гущи с интеграцией в Telegram Mini Apps. Платформа включает систему подписок, платежные интеграции, AI-анализ изображений и персонализацию результатов.

**Технологический стек:**
- Backend: Node.js (Fastify), PostgreSQL, Redis, BullMQ
- Frontend: React + TypeScript (Telegram Mini App)
- AI: OpenRouter (Vision + Text models)
- Инфраструктура: Railway, Supabase
- Платежи: Telegram Stars, СБП, CloudPayments

## 4. ПОРЯДОК СДАЧИ-ПРИЕМКИ РАБОТ

### 4.1. Процедура приемки этапа
1. Исполнитель уведомляет заказчика о готовности этапа (email + демонстрация)
2. Заказчик проводит тестирование в течение **7 рабочих дней**
3. Заказчик направляет:
   - **Акт приемки** (при успешной приемке), либо
   - **Мотивированный отказ** с перечнем замечаний
4. Исполнитель устраняет замечания в течение **5 рабочих дней**
5. После устранения замечаний - повторная приемка (пп. 2-4)

### 4.2. Критерии успешной приемки
- Реализован весь функционал, указанный для этапа (раздел 2)
- Отсутствуют критические ошибки (блокирующие основной функционал)
- Система работает стабильно в тестовой среде
- Предоставлена базовая документация по развернутому функционалу
